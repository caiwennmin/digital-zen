# 🧘 Digital Zen · 数字禅

> *在信息洪流中，找回你的深度专注力。*

**数字禅** 是一个为数字时代设计的正念生产力伴侣。在 AI 生成内容泛滥、注意力成为新货币、数字成瘾无声蔓延的当下 —— 数字禅帮助你暂停、呼吸，专注于真正重要的事。

---

## 🌍 为什么有这个项目

我们生活在一个前所未有的挑战时代：

| 挑战 | 影响 |
|------|------|
| **AI 内容过载** | 越来越难分辨真伪信息 |
| **注意力碎片化** | 平均注意力广度降至 8 秒以下 |
| **数字成瘾** | 社交媒体算法利用多巴胺回路 |
| **心理健康危机** | 焦虑与倦怠率达到历史新高 |
| **社交孤立** | 高度连接却前所未有的孤独 |

数字禅是一次小小的抵抗 —— 一个帮助你夺回注意力和身心健康主动权的小工具。

---

## ✨ 功能

- **🍅 专注计时器** — 番茄钟风格，自定义工作/休息时长
- **🌿 环境白噪音** — 基于 Web Audio API 生成的雨声、森林、咖啡馆音效（无需下载）
- **📝 每日意图** — 每天早晨设定 3 个优先级 + 晚间反思
- **📊 进度仪表盘** — 追踪专注会话、总时长、连续天数、周趋势图
- **🔒 隐私优先** — 所有数据仅存储在本地浏览器。无追踪、无服务器、无需注册
- **🎨 禅意美学** — 暗色主题、舒缓渐变、流畅动画
- **📱 响应式设计** — 桌面端和移动端完美适配

---

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/digital-zen.git
cd digital-zen

# 用浏览器打开（无需构建！）
open index.html
# 或直接双击 index.html
```

就这些。不需要 `npm install`，没有依赖，没有构建工具。纯 HTML、CSS 和原生 JavaScript。

## 🏗 项目结构

```
digital-zen/
├── index.html          # 主应用入口
├── css/
│   └── main.css        # 所有样式（CSS 变量、响应式、动画）
├── js/
│   ├── app.js          # 应用控制器与导航
│   ├── timer.js        # 番茄钟计时器
│   ├── journal.js      # 每日意图与反思
│   ├── stats.js        # 统计仪表盘与图表
│   ├── sounds.js       # Web Audio API 环境音生成器
│   └── storage.js      # localStorage 封装与连续天数追踪
├── assets/             # 静态资源目录
├── README.md           # 英文说明
├── README_CN.md        # 本文件
└── LICENSE             # MIT 许可证
```

## 🛠 技术栈

- **原生 JavaScript** — 无框架、零依赖

- **localStorage** — 保护隐私的数据持久化
- **SVG** — 带动画的进度环
