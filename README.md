# 🧘 Digital Zen · 数字禅

> *In the flood of information, reclaim your deep focus.*

**Digital Zen** is a mindful productivity companion designed for the modern digital age. In a world where AI-generated content floods our feeds, attention is the new currency, and digital addiction is the silent epidemic — Digital Zen helps you pause, breathe, and focus on what truly matters.

---

## 🌍 Why This Project Exists

We live in an era of unprecedented challenges:

| Challenge | Impact |
|-----------|--------|
| **AI Content Overload** | Harder to distinguish signal from noise |
| **Attention Fragmentation** | Average focus span dropped below 8 seconds |
| **Digital Addiction** | Social media algorithms exploit dopamine loops |
| **Mental Health Crisis** | Anxiety & burnout rates at all-time highs |
| **Social Isolation** | Hyper-connected yet lonelier than ever |

Digital Zen is a small act of resistance — a tool to help you reclaim agency over your attention and well-being.

---

## ✨ Features

- **🍅 Focus Timer** — Pomodoro-style timer with customizable work/break intervals
- **🌿 Ambient Sounds** — Web Audio API generated rain, forest, and café sounds (no downloads needed)
- **📝 Daily Intentions** — Set 3 priorities each morning + evening reflection
- **📊 Progress Dashboard** — Track focus sessions, total minutes, streaks, and weekly trends
- **🔒 Privacy First** — All data stored locally in your browser. No tracking, no servers, no accounts.
- **🎨 Zen Aesthetic** — Dark theme with calming gradients and smooth animations
- **📱 Responsive** — Works beautifully on desktop and mobile

---

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/digital-zen.git
cd digital-zen

# Open in browser (no build step needed!)
open index.html
# or just double-click index.html
```

That's it. No `npm install`, no dependencies, no build tools. Pure HTML, CSS, and vanilla JavaScript.
