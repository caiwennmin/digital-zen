# 🧘 Digital Zen · 数字禅

> *In the flood of information, reclaim your deep focus.*

**Digital Zen** is a mindful productivity companion designed for the modern digital age. In a world where AI-generated content floods our feeds, attention is the new currency, and digital addiction is the silent epidemic — Digital Zen helps you pause, breathe, and focus on what truly matters.


---

## 🔐 Account & Cloud Sync

Digital Zen now supports account login with cloud data sync.

### Quick Start (with server)

```bash
cd server
npm install
npm start
```

Then open **http://localhost:3456** in your browser.

The server provides:
- **Email/password registration & login** (JWT, bcrypt)
- **Cloud data sync** — your focus sessions, journal entries, and stats sync automatically
- **Cross-device access** — log in on any device to see your data
- **Zero external dependencies** — SQLite database stored locally, no cloud services needed

### Without server

Just double-click `index.html` — everything works fully offline with localStorage. Login features require the server.

## 🔐 账号与云同步

支持邮箱注册/登录，数据云端同步。

### 启动服务器

```bash
cd server
npm install
npm start
```

浏览器打开 **http://localhost:3456**

功能：
- **邮箱注册登录** — JWT + bcrypt 加密
- **云端数据同步** — 专注记录、日记、统计自动同步
- **跨设备访问** — 任何设备登录即可看到自己的数据
- **零外部依赖** — SQLite 本地数据库，不需要任何云服务

### 不用服务器

直接双击 `index.html`，纯离线使用 localStorage。登录功能需要启动服务器。
